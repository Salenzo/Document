\section{逻辑门}

\begin{enumerate}

\item 与门

符号：$^A_B{\large=}\!\boxed{\&}\!\!-\!\!{\small F}$

表达式：$F=A \cdot B$

\item 或门

符号：$^A_B{\large=}\!\boxed{\geq{1}}\!\!-\!\!{\small F}$

表达式:$F=A+B$

\item 非门

符号：$A{\large-}\!\boxed{1}{\normalsize\circ}\!\!-\!\!{\small F}$

表达式:$F=\overline A$

\item 复合逻辑门

\begin{enumerate}

    \item 与非门

          符号:$^A_B{\large=}\!\boxed{\&}{\normalsize\circ} \!\!-\!\!{\small F}$

          表达式:$F=\overline{AB}$

    \item 或非门

          符号:$^A_B{\large=}\!\boxed{\geq{1}}{\normalsize\circ} \!\!-\!\!{\small F}$

          表达式:$F=\overline{A+B}$

    \item 与或非门

          表达式:$F=\overline{AB+CD}$

    \item 异或门

          符号:$^A_B{\large=}\!\boxed{={1}} \!\!-\!\!{\small F}$

          表达式:$F=A\oplus{}B=A\overline{B}+\overline{A}B$

          同或运算:$F=A\odot{}B=AB+\overline{AB}$

\end{enumerate}

\section{竞争与冒险}

原因:信号传输延迟

用输入、输出时序图表示

\subsection{竞争}

竞争:输入信号通过不同途径达到输出端的时间不同(随机过程)

\begin{enumerate}

    \item 非临界竞争:不会产生错误
    \item 临界竞争:导致逻辑错误

\end{enumerate}

\begin{enumerate}

    \item 代数判别法

          从\textbf{函数表达式结构}判别

          \begin{enumerate}

              \item 如果某变量同时以原变量反变量形式存在
              \item 将其他变量可能的取值代入
              \item 如果出现$x+\overline{x}$或$x\overline{x}$则可能产生冒险

          \end{enumerate}

    \item 卡诺图判别法

          \begin{enumerate}

              \item 画出各**与项**对应卡诺圈
              \item 如果两卡诺圈**相切**(存在共用的相邻最小项)则可能产生冒险
          \end{enumerate}

\end{enumerate}
