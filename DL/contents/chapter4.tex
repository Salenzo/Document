\section{触发器}

\begin{enumerate}
    \item 储存电路
    \item 能储存一位二进制数
    \item 在任一时刻只处于一种\textbf{稳态}
\end{enumerate}

\subsection{R-S 触发器}

\begin{enumerate}
    \item 基本型(锁存器)
    \item 直接 复位-置位
    \item 组成：
    \item 交叉耦合或非门
    \item 交叉耦合与非门
    \item 时钟型
    \item 4\*与非
\end{enumerate}

空翻现象：由于时钟信号宽度而多次翻转，可由主从触发器(串联)解决。

\subsection{D 触发器}

    \item 输入信号转换为互补信号
